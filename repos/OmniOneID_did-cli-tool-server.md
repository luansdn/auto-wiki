## 项目介绍

OmniOneID/did-cli-tool-server 是一款基于 Java 开发的分布式身份标识（DID）服务端 SDK 工具。 该工具专注于服务端钱包与 DID 文档的自动化生成与管理，提供完整的命令行接口与标准化 API，支持密钥对生成、凭证签发、文档验证及多格式 DID 文档解析。其采用模块化架构设计，兼容主流 DID 方法，并严格遵循 Apache-2.0 开源协议，确保代码的安全性与可拓展性。 适用于需要快速集成去中心化身份认证体系的区块链开发者、企业级身份管理平台架构师以及 Web3 应用团队，可广泛应用于数字身份管理、跨链身份互认及合规性审计等场景。


## 主要特性

• 支持DID标识符的生成、解析与生命周期管理 • 提供可验证凭证（VC）的签发、验证与撤销功能 • 内置DID文档的存储、查询与版本控制机制 • 集成区块链底层实现去中心化身份数据锚定 • 提供命令行交互界面与RESTful API双模调用 • 支持国密算法与国际标准加密协议保障通信安全 • 模块化架构设计便于跨平台部署与二次开发 • 兼容多链与跨域身份互操作协议标准
