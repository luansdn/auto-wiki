## 项目介绍

wp-cli/php-cli-tools 是一个专为 PHP 开发者打造的命令行工具库，旨在简化终端脚本的开发流程。


## 主要特性

• 提供丰富的终端输出格式化功能（颜色、样式、加粗等） • 支持交互式命令行输入与提示符处理 • 内置进度条组件，便于显示任务执行状态 • 支持表格渲染，可整齐输出结构化数据 • 自动检测终端宽度与平台兼容性（跨Windows/Linux/macOS） • 提供简洁易用的API，便于快速构建命令行工具 • 支持UTF-8编码与国际字符显示
