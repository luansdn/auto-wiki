## 项目介绍

hsweb-framework 是一款基于 Spring Boot 2.x 构建的开源企业级后台管理系统基础框架。 该框架率先采用全响应式编程架构，深度融合 Reactor、Spring WebFlux 与 R2DBC 技术，实现高并发下的异步非阻塞处理。内置完善的权限管理、数据权限、动态表单及工作流模块，提供开箱即用的开发体验与高度可扩展的插件化设计。 (Core features: reactive programming, high concurrency, built-in modules like RBAC, data permission, dynamic forms, plugin architecture) 适用于需要快速搭建高性能后台系统、微服务架构或高并发数据交互场景的企业级开发团队。主要面向追求现代化技术栈、注重系统响应速度与资源利用率的中高级 Java 开发者及架构师。 (Scenarios: high-performance backend, microservices, high concurrency; Audience: enterprise dev teams, mid/senior Java devs, architects) hsweb-framework 是一款基于 Spring Boot 2.x 构建的开源企业级后台管理系统基础框架。(38) 该框架率先采用全响应式编程架构，深度融合 Reactor、Spring WebFlux 与 R2DBC 技术，实现高并发下的异步非阻塞处理。内置完善的权限管理、数据权限、动态表单及工作流模块，提供开箱即用的开发体验与高度可扩展的插件化设计。(118) 适用于需要快速搭建高性能后台系统、微服务架构或高并发数据交互场景的企业级开发团队。主要面向追求现代化技术栈、注重系统响应速度与资源利用率的中高级 Java 开发者及架构师。(88)


## 主要特性

• 基于Spring Boot构建，轻量级快速开发框架 • 内置RBAC权限管理系统，支持细粒度角色与数据权限控制 • 提供可视化代码生成器，一键生成前后端基础代码 • 采用模块化架构设计，支持按需引入功能组件 • 内置多数据源管理与动态数据源切换功能 • 集成国际化支持与多语言切换机制 • 提供丰富的后台管理UI组件与主题定制能力 • 完善的文档体系与官方示例项目，降低上手门槛
