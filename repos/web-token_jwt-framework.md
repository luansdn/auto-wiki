## 项目介绍

JWT Framework 是一款基于 PHP 语言开发的开源 JSON Web Token 处理库，全面支持 JOSE（JSON Object Signing and Encryption）标准。 该框架提供完整的 JWT 签名、验证、加密与解密功能，涵盖 JWS、JWE、JWK 及 JWKSet 等核心规范。其设计注重安全性与灵活性，内置多种加密算法支持，并采用模块化架构便于集成与扩展，同时严格遵循现代密码学最佳实践。 适用于需要实现安全身份认证、数据完整性校验及敏感信息传输的 PHP 后端应用。主要面向后端开发者、安全工程师及需要快速构建标准化 JWT 交互系统的团队，是构建微服务架构与 API 网关的理想选择。


## 主要特性

• 模块化架构设计，支持按需引入核心组件与算法库 • 完整遵循JWT国际标准（RFC 7519、JWS、JWE、JWK等） • 支持多种加密与签名算法（RSA、ECDSA、HMAC、EdDSA等） • 提供完整的Token生成、解析、验证与加解密功能 • 兼容PSR-7与PSR-15标准，便于无缝集成现代PHP框架 • 内置JWK密钥管理与密钥集（JWK Set）自动刷新机制 • 高度可扩展，支持自定义算法、签名器、验证器与HTTP中间件
